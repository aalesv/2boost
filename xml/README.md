### defs_template.xml

General template for RomRaider definitions. Contains all possible tables, so please remove unneeded.
Please take note that defs for newer ROMs (that use Requested Torque) usually should use boost related table
names with space at the end, like "Target Boost Overtake map ".

### RR_2BOOST.xml

RomRaider definitions for supported 2Boost ROMs.

### target\*.xml

RomRider definitions template for particular CALID. Needed to generate definitions when buildind a ROM.
